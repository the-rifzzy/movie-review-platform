# Movie Review and Rating Platform

A web application to browse, review, and rate movies. Users can view detailed movie information, submit reviews, and give star ratings.

## Project Overview
This project allows users to:  
- Browse popular movies in a card-based layout.  
- View detailed information and reviews.  
- Submit their own reviews with star ratings.  
- See previously submitted reviews and ratings.  

It is built using a **Node.js backend** with REST APIs and a **React.js frontend** for a responsive, interactive experience.

## Features
- Movie browsing with card layout   
- Review submission with star ratings  
- Display of existing reviews   

## Requirements
- Node.js (v18 or higher recommended)  
- npm (Node Package Manager)  
- MongoDB (if using a database)  

## How to Run
1. Clone the repository
git clone https://github.com/the-rifzzy/movie-review-platform.git
cd movie-review-platform
2. Install dependencies: `npm install`
3. Start server: `node server.js`
4. Open your browser at: http://localhost:3000
5. Username:admin, Password:1234
